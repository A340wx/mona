GUIサーバーについて

1. 特徴

・Mona におけるデファクトスタンダードな GUI サーバーです
・マルチウィンドウを実現しています
・キーボード、マウスをサポートしています
・UTF-8に対応しています (日本語もOK)
・プロポーショナルフォント表示に対応しています
・24ビットビットマップファイルに対応しています
・JPEGファイルに対応しています
・部品ごとにダブルバッファリングを行っています
・設定ファイルで挙動をカスタマイズできます
・非常に単純なプロトコルなので独自の GUI を作りやすいです
・X Window System との互換性はありませんが非常に小さいです

2. 制限事項

・core から libbz2 がはずされたため、bzip2 で圧縮された画像は扱えません (USE_BZIP2)
・MONAWALL.BM5 は 800x600x24 ビット BMP を tek5 圧縮したものです
  自分の好みの画像に差し替えることが可能です
・MONA12.MF5 はユニコード化した 12 ドットモナーフォントを tek5 圧縮したものです
  自分の好みのフォントに差し替えることは不可能ではありませんがかなり面倒です
・libgui との依存性がなくなるように修正している個所が何箇所かあります
