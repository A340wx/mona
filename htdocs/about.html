About

Small, new and fast micro kernel operationg system written in C++.

  • MonaOS is neither POSIX nor Windows clone.
  • Boot from floppy disk or CD-ROM.
  • Kernel size is now 98KB(2006/08/19)

Requirements

  • Intel architecture 32(IA32)
